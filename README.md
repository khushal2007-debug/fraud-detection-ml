# 💳 Credit Card Fraud Detection

AI Powered Real-time Fraud Detection System

## 🎯 What it does
- Detects fraudulent credit card transactions in real-time
- 99.91% accuracy on 284,807 real transactions
- FastAPI backend + Streamlit UI

## 🛠️ Tech Stack
- Python
- Scikit-learn (Logistic Regression)
- FastAPI
- Streamlit
- Pandas & NumPy

## 📊 Dataset
- 284,807 real transactions
- 492 actual fraud cases
- Source: Kaggle MLG-ULB

## ▶️ How to Run

FastAPI:
uvicorn main:app --reload

Streamlit:
streamlit run app.py
