# 💳 Credit Card Fraud Detection System

> AI Powered Real-time Fraud Detection using Machine Learning + FastAPI + Streamlit

---

## 🎯 What is this project?

This system detects fraudulent credit card transactions in real-time using Machine Learning.
Banks can integrate this directly via REST API to automatically block suspicious transactions.

---

## 🚀 How it Works
```
Customer makes a transaction
        ↓
Transaction data sent to FastAPI
        ↓
Random Forest Model analyzes it
        ↓
FRAUD or SAFE decision in milliseconds
        ↓
Bank blocks or approves transaction
```

---

## 📊 Model Performance

| Metric | Score |
|--------|-------|
| Accuracy | 99.91% |
| Dataset | 284,807 real transactions |

## ▶️ How to Run

### Step 1 — Install Dependencies
```bash
pip install -r requirements.txt
```

### Step 2 — Download Dataset
Download `creditcard.csv` from:
🔗 kaggle.com/datasets/mlg-ulb/creditcardfraud

### Step 3 — Train Model
Run `fraud_detection.ipynb` — this generates `fraud_model.pkl`

### Step 4 — Start FastAPI
```bash
uvicorn main:app --reload
```

### Step 5 — Start Streamlit
```bash
streamlit run app.py
```

## 🔌 API Usage

### Endpoint
```
POST http://localhost:8000/predict
```

### Request
```json
{
  "features": [0.0, -1.35, -0.07, 2.53, 1.37, -0.33, 
               0.46, 0.23, 0.09, 0.36, 0.09, -0.55, 
               -0.61, -0.99, -0.31, 1.46, -0.47, 0.20, 
               0.02, 0.40, 0.25, -0.01, 0.27, -0.11, 
               0.06, 0.12, -0.18, 0.13, -0.02, 149.62]
}
```

### Response
```json
{
  "prediction": 0,
  "result": "SAFE",
  "fraud_probability": 0.02,
  "safe_probability": 99.98
}
```

```

                    SAFE  = Approve transaction
                                  ↓
                    Customer gets SMS alert
```
